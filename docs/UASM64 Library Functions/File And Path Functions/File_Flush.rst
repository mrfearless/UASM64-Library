.. _File_Flush:

==========
File_Flush
==========

Flushes the file buffers to disk.

::

   File_Flush PROTO hFile:QWORD


**Parameters**

* ``hFile`` - handle of the opened file to flush.


**Returns**

TRUE if successful, or FALSE otherwise.
