===================
CPU_AESNI_Supported
===================

Check if Advanced Encryption Standard New Instructions (AES-NI) instructions are supported.

::

   CPU_AESNI_Supported PROTO 


**Parameters**


**Returns**

TRUE if AES-NI instructions are supported, or FALSE otherwise.
