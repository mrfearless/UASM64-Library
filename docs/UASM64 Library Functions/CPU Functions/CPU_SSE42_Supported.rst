===================================
CPU_SSE42_Supported 
===================================

Check if Streaming SIMD Extensions 4.2 (SSE42) instructions are supported.
    
::

   CPU_SSE42_Supported PROTO


**Parameters**

There are no parameters.


**Returns**

``TRUE`` if SSE42 instructions are supported, or ``FALSE`` otherwise.

**Notes**

`https://en.wikipedia.org/wiki/SSE4 <https://en.wikipedia.org/wiki/SSE4>`_

**Example**

::

   Invoke CPU_SSE42_Supported
