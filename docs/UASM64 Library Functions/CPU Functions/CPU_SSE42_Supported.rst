CPU_SSE42_Supported
===================

Check if Streaming SIMD Extensions 4.2 (SSE42) instructions are supported.

::

   CPU_SSE42_Supported PROTO 


**Parameters**


**Returns**

TRUE if SSE42 instructions are supported, or FALSE otherwise.
