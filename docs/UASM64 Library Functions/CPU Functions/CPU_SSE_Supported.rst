CPU_SSE_Supported
=================

Check if Streaming SIMD Extensions (SSE) instructions are supported.

::

   CPU_SSE_Supported PROTO 


**Parameters**


**Returns**

TRUE if SSE instructions are supported, or FALSE otherwise.
