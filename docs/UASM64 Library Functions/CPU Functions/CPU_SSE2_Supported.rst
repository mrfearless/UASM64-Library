CPU_SSE2_Supported
==================

Check if Streaming SIMD Extensions 2 (SSE2) instructions are supported.

::

   CPU_SSE2_Supported PROTO 


**Parameters**


**Returns**

TRUE if SSE2 instructions are supported, or FALSE otherwise.
