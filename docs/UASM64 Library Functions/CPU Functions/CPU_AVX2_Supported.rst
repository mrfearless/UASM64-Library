==================
CPU_AVX2_Supported
==================

Check if Advanced Vector Extensions 2 (AVX2) instructions are supported.

::

   CPU_AVX2_Supported PROTO 


**Parameters**


**Returns**

TRUE if AVX2 instructions are supported, or FALSE otherwise.
