=================
CPU_AVX_Supported
=================

Check if Advanced Vector Extensions (AVX) instructions are supported.

::

   CPU_AVX_Supported PROTO 


**Parameters**


**Returns**

TRUE if AVX instructions are supported, or FALSE otherwise.
