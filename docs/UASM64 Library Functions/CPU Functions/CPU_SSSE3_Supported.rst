===================
CPU_SSSE3_Supported
===================

Check if Supplemental Streaming SIMD Extensions 3 (SSSE3 or SSE3S) instructions are supported.

::

   CPU_SSSE3_Supported PROTO 


**Parameters**


**Returns**

TRUE if SSSE3 instructions are supported, or FALSE otherwise.
