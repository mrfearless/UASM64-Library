CPU_SSE41_Supported
===================

Check if Streaming SIMD Extensions 4.1 (SSE41) instructions are supported.

::

   CPU_SSE41_Supported PROTO 


**Parameters**


**Returns**

TRUE if SSE41 instructions are supported, or FALSE otherwise.
